Assessment Interoperability Framework Component Metadata Records

This is a repository to store the DCAT metadata records for various components of the Assessment Interoperability Framework defined by OSTrails.

This repository is used, by default, by the DSW Authoring Tool, when a metadata record is completed; however, it can be used by anyone through obtaining an account, or by forking and making a pull-request.

These metadata records are served from subfolders as follows:
  • Benchmarks
  • Metrics
  • Algorithms
  • Tests
